# Chrome Web Store Listing

## Basic Information

**Name:** Tailwind Color Picker

**Short Description (132 characters max):**
Pick colors from any webpage and get the closest Tailwind CSS class name. Fully offline & private.

**Detailed Description:**
Pick colors from any webpage and instantly find the closest matching Tailwind CSS class name. Fully offline, private, and built with Manifest V3.

**Features:**
- EyeDropper API integration for color picking
- CIE Lab color space matching with DeltaE calculation
- Copy Tailwind class names (e.g., `bg-blue-500`) or HEX codes
- Keyboard shortcut: `Alt+C` to open picker window
- Last picked color is saved locally
- 100% offline operation, no network requests
- Dark-themed developer-friendly UI

**Usage:**
1. Click the extension icon or press `Alt+C`
2. Click "Pick Color" to activate the eyedropper
3. Click anywhere on screen to sample a color
4. Copy the Tailwind class or HEX code

**Privacy:**
- All processing happens locally
- No network requests
- No analytics or tracking
- Only stores last picked color locally
