# Privacy Policy for Tailwind Color Picker

**Last updated:** December 18, 2025

## Overview

Tailwind Color Picker is a Chrome extension that helps developers pick colors from webpages and find matching Tailwind CSS class names. This privacy policy explains how we handle user data.

## Data Collection

**We do not collect, store, or transmit any personal data.**

The extension operates entirely offline and processes all data locally in your browser.

## Local Storage

The extension uses Chrome's local storage API to save your last picked color. This data:
- Is stored only on your device
- Never leaves your browser
- Is not transmitted to any external servers
- Can be cleared at any time through Chrome's extension settings

## Permissions

The extension requires the following permission:
- **storage**: Used only to save your last picked color locally for convenience

## Third-Party Services

This extension does not:
- Use any analytics services
- Send data to external servers
- Include third-party tracking code
- Make network requests
- Use CDN resources

## Data Sharing

We do not sell, rent, or share any user data with third parties because we do not collect any data.

## Contact

## Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated revision date.
